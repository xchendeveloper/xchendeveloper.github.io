## 网关分类
 - 排他网关
 - 并行网关
 - 包容网关
 - 事件网关

### 排他网关
和条件顺序流配合使用，执行到改网关时，按照输入流的顺序逐个计算，条件计算结果为true时，继续执行当前网关输出流。

如果多个线路计算结果都为true，取第一个。
如果都不为true,殷勤抛出异常。

```
<exclusiveGateway id="exclusivegateway1" name="Exclusive Gateway" default="flow2"></exclusiveGateway>
```

如果指定了default,则都不为true时执行指定的输出流
